1 Objetivo

Estimar a prevalência e características sociodemográficas associadas à insegurança alimentar (IA).

2 Métodos

As análises utilizaram dados da terceira onda do Estudo Longitudinal da Saúde dos Idosos Brasileiros (ELSI-Brasil). Mais informações sobre o estudo estão disponíveis no site do ELSI-Brasil.

4 Organização das variáveis

A amostra inicial incluiu 10.773 indivíduos. Foram excluídos 28 participantes com resposta “não sabe/não respondeu” para IA e 2.197 indivíduos com menos de 60 anos de idade, resultando em uma amostra analítica final de 8.548 participantes.

6 Prevalência IA

A prevalência de IA na população estudada foi de 4,7% (IC95%: 3,4–6,5).

Prevalência IA 
prev_ia <- svyciprop(
  ~ I(n73_6 == "Sim"),
  desenho,
  method = "logit"
)
prev <- coef(prev_ia)[1] * 100
ic <- confint(prev_ia) * 100
tibble(
  Indicador = "Insegurança alimentar",
  `Prevalência (%)` = round(prev, 1),
  `IC95%` = sprintf(
    "%.1f–%.1f",
    ic[1],
    ic[2]
  )
)

calc_prev <- function(var, label, desenho) {
  prev <- svyby(
    ~ I(n73_6 == "Sim"),
    as.formula(paste0("~", var)),
    desenho,
    svyciprop,
    vartype = "ci",
    method = "logit",
    na.rm = TRUE
  )
  p_val <- svychisq(
    as.formula(paste0("~n73_6 + ", var)),
    desenho
  )$p.value
  categoria <- as.character(prev[[var]])
  tibble(
    Caracteristica = c(label, paste0("  ", categoria)),
    `Prevalência (%)` = c(
      "",
      sprintf("%.1f", 100 * prev[[2]])
    ),
    `IC95%` = c(
      "",
      sprintf(
        "%.1f–%.1f",
        100 * prev$ci_l,
        100 * prev$ci_u
      )
    ),
    `p-valor` = c(
      format.pval(
        p_val,
        digits = 3,
        eps = 0.001
      ),
      rep("", length(categoria))
    ),
    grupo = c(TRUE, rep(FALSE, length(categoria)))
  )
}

      OR_IC95 = "",

        "<0.001",
        sprintf("%.3f", p_global)

      destaque = p_global < 0.20
    ),

      Caracteristica = paste0("  ", ref, " (ref.)"),

          paste0("^", v),

        res$estimate,
        res$conf.low,
